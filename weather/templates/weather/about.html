Welcome to the website!


This is an application for helping user to check weather. You can check weather in every city around the world. I used Django for backend and Bootstrap for the frontend. And also I used API of OpenWeatherMap to check weather and I consumed the API with python package requests.




How to use this app?


1. You have to add a new city for check the weather
2. If the city doesn't exist or city is already in the list the app will be give you some notification
3. You can delete the city from the list
